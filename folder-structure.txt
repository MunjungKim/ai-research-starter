Create this folder structure in the current directory:

CLAUDE.md              # context & rules — the key file
.claude/               # settings + custom skills
seeds/                 # seed papers & sources
data/raw/              # original data — never modify
drafts/                # writing (.tex .md .docx)
papers/[journal name]/figures/   # one folder per target journal → its manuscript + figures
workflow/              # numbered pipeline — run in order
workflow/scripts/      # reusable helper scripts
notebook/              # numbered analysis notebooks
notes/                 # project hub — ongoing thinking as you work
notes/ideas/           # one file per research idea
notes/dashboard.html   # live project dashboard
notes/meetings/        # meeting notes
notes/updates/         # progress updates
results/               # large outputs (heavy files)
deprecated/ · trash/   # archive & soft-delete — never rm
pyproject.toml · uv.lock   # Python env via uv
